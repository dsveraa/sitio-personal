Divido el proyecto en distintas etapas que ayudan a avanzar sobre una base sólida, partiendo con un flujograma para visualizar las conexiones y comportamientos de interacción, luego un diseño en wireframe para definir la diagramación del contenido, maqueta en HTML/CSS/Bootstrap responsiva adaptable a diferentes pantallas para luego implementar el diseño final y funcionalidades del sitio mediante JavaScript o Typescript.

I divide the project into different stages that help to progress on a solid foundation, starting with a flowchart to visualize the connections and interaction behaviors, then a wireframe design to define the layout of the content, followed by a responsive HTML/CSS/Bootstrap mockup adaptable to different screens, and finally, implementing the final design and functionalities of the site using JavaScript or TypeScript.
